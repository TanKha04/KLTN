# Hướng dẫn upload ảnh lên Overleaf

## Bước 1: Tạo thư mục images trên Overleaf
1. Trong Overleaf, click vào biểu tượng "New Folder" (thư mục mới)
2. Đặt tên thư mục là `images`

## Bước 3: Chuyển đổi file SVG và WEBP sang PNG

### Cách 1: Sử dụng công cụ online
- SVG to PNG: https://svgtopng.com/
- WEBP to PNG: https://cloudconvert.com/webp-to-png

### Cách 2: Sử dụng Inkscape (cho SVG)
1. Mở file SVG trong Inkscape
2. File > Export PNG Image
3. Chọn kích thước và lưu

### Cách 3: Sử dụng Paint hoặc GIMP (cho WEBP)
1. Mở file WEBP
2. Save As > PNG

## Bước 4: Compile báo cáo
Sau khi upload tất cả ảnh, click "Recompile" trong Overleaf để xem kết quả.

## Lưu ý quan trọng:
- Overleaf KHÔNG hỗ trợ file SVG trực tiếp, phải chuyển sang PNG
- Overleaf KHÔNG hỗ trợ file WEBP, phải chuyển sang PNG
- Tên file không được có dấu tiếng Việt
- Tên file phân biệt chữ hoa/thường
